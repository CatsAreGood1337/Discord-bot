# Discord Bot

### Установка:
##### 1. Установить нужные библиотеки
- Скопируйте репозиторий или скачайте его 
- Откройте консоль/IDE и пропишите `pip install -r requirements.txt`
- Вставьте бот токен в файл "config.py"
- Запустите файл `start.bat`
